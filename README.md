# Dream Scape 

A simple website that allows users to type in their dreams and get a photos related of them.

## Features
* Interactive UI: Smooth style.
* Floating Animation: Key words drift across the viewport while generating.
* Visual Representation: Generates a link to look at a photo related to your dream.
* Responsive Layout: Automatically locks perfectly to the center of any browser frame.

## Technologies Used
* HTML
* CSS
* JavaScript
