# DreamScape 

DreamScape is an artistic web experience that transforms dreams, thoughts, and emotions into animated visual scenes using HTML, CSS, and JavaScript.

Users can type a dream, feeling, or idea, and the website creates a cinematic atmosphere with floating particles, glowing animations, animated text, and immersive visuals.

---

## Features

- Animated dream visualization
- Floating particles and stars
- Cinematic UI design
- Dynamic animated text
- Emotion-based atmosphere
- Smooth transitions and effects
- Responsive design
- Background music support
- Save dreams locally using localStorage

---
